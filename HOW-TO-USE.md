## What is Markmap?

Markmap converts Markdown into an interactive mind map. Nodes expand and collapse on click, you can zoom with your scroll wheel, and pan by dragging the canvas. It's entirely free and runs in a browser or VS Code.

## Method 2 â€” VS Code Extension (Best for Study)

### Install
1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for **Markmap** by `gera2ld`  
   Extension ID: `gera2ld.markmap-vscode`
4. Click Install

### Use
1. Open any `.md` file from this repo in VS Code
2. Press `Ctrl+Shift+P` to open the Command Palette
3. Type `Markmap: Open as Markmap` and hit Enter
4. A live-updating map opens in a split panel

### Study workflow tip
- Put the map on the right panel (click and drag the tab)
- Open a blank `.md` for notes on the left
- As you study each node, write your own notes on the left in your own words
- The act of restating things builds retention faster than re-reading
